Specific Uses of Tabular Data

  • Periodic Table
  • Spreadsheet
  • Logic of Truth
  • Mathematical Table

Periodic Table

a table of the chemical elements arranged in order of atomic number, usually in rows, so that elements with similar atomic structure (and hence similar chemical properties) appear in vertical columns.

Spreadsheet

An electronic document in which data is arranged in the rows and columns of a grid and can be manipulated and used in calculations.

Logic truth table

This tool generates truth tables for propositional logic formulas.(specifically in connection with Boolean algebra, boolean functions, and propositional calculus)

Mathematical Table

A multiplication table (sometimes, less formally, a times table) is a mathematical table used to define a multiplication operation for an algebraic system.
